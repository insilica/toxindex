# Ranking Workflow: Outputs and Artifacts

This page describes the outputs and artifacts generated by the Ranking workflow.

## Primary Outputs

### Ranking Results

**Overall Rankings:**
- Numerical ranking positions (1, 2, 3, ...)
- Chemical identifiers and names
- Ranking scores and confidence intervals
- Statistical significance indicators

**Pairwise Comparisons:**
- Head-to-head comparison matrices
- Win/loss records for each chemical
- Statistical significance of comparisons
- Confidence intervals for pairwise rankings

### Statistical Analysis

**Confidence Intervals:**
- Bootstrap-based confidence intervals
- Bayesian credible intervals
- Sensitivity analysis results
- Uncertainty quantification

**Significance Testing:**
- P-values for ranking differences
- Multiple comparison corrections
- Effect size calculations
- Power analysis results
