# Getting Started

Follow these steps to run RAP on ToxIndex.

## 1. Open ToxIndex

Go to `https://www.toxindex.com/`.

## 2. Pick a workflow

Use the workflow selector and choose RAP.

![Tool selection](/screenshot_app/tool_selection.png)

## 3. Submit your query

- Enter a prompt or upload a file if required
- Click Submit

![Submit query](/screenshot_app/submit_query.png)

## 4. Monitor the task

You will see the task running in the dashboard.

![Task running](/screenshot_app/task_running.png)

## 5. View results

Open the finished task to view outputs and artifacts.

![Task details](/screenshot_app/task_details.png)
