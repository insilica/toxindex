# Overview

RAP (Rapid Assessment Pipeline) provides a quick toxicity assessment workflow.

![Landing](/screenshot_app/landing_page.png)

## What RAP does

- Ingests your prompt or file input
- Runs the RAP analysis pipeline
- Produces summarized results and artifacts

## When to use RAP

- Early exploration and triage
- Summarizing toxicity evidence
- Creating shareable outputs
